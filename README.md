# SecretsBlog

SecretsBlog is an anonymous secrets sharing blog. It uses LocalStorage to save all the records that are created and allows you to edit or delete them.

## Features
* View all records
* Create a new record
* Edit an existing record
* Delete record

## Built With
* Ruby version - 2.7.2
* Rails version - 6.0.3
* Database sqlite3 version - 1.4
